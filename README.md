## cara install aplikasi
1. Masukkan perintahcomposer install
2. Ubah folder .env.example menjadi .env
3. Pada file .env atur nama database menhadi db_laravel_uas
4. Import database yang ada di folder database ke php myadmin/sql yoq
5. Setelah itu jalankan perintah php artisan serve
